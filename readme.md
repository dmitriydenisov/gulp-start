# Сборка GULP4 v2

### Перед использованием установить:

#### GULP глобально и локально
#### npm install webp-converter@2.2.3 --save-dev

1. Шаблонизатор PUG
2. Препроцессор SCSS

7. Задачу переноса подключаемых библиотек
8. Задачу очистки сборки проекта
